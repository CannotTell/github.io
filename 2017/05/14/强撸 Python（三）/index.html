Python的多态理解

背景

面向对象中有三大特性： 封装，继承和多态

静态语言的多态

我们先看2段代码，一段是静态语言Java。
在Java里放宽了讲其实有2中多态形势：

  • 编译时多态（设计时多态）：方法重载，方法名称相同，参数个数、次序、类型不同，返回值也可以不同
  • 运行时多态（后绑定）：我们讲的主要是这种
    Java里多态的存在要有三个前提：

1.要有继承关系
2.子类要重写父类方法
3.父类引用指向子类对象

首先Python是一门动态语言
